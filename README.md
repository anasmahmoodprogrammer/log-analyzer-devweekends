# Log Analyzer

A robust CLI tool built in Node.js to parse web server logs, handle format deviations gracefully, and output an actionable summary report.

## Prerequisites
- Node.js (v14.x or higher) installed on your machine.
- No external npm packages are required. The tool relies entirely on native Node.js modules (`fs`, `readline`, `path`).
